介绍

JavaScript 事件是前端开发中至关重要的概念之一，它使得网页可以对用户的操作做出响应，实现交互性和动态性。在本节中，我们将详细介绍 JavaScript 事件的概念、类型、以及如何使用它们来实现各种交互效果。

什么是事件？

事件是指用户在网页上执行的动作，比如点击、鼠标移动、键盘输入等。JavaScript 通过事件监听器来捕获这些动作，并执行相应的代码，从而实现网页的交互功能。

JavaScript 事件的类型

JavaScript 鼠标事件

用户与页面的交互经常涉及到鼠标操作，比如点击、移动、滚动等。

  • click: 当鼠标点击某个元素时触发。
  • mouseover / mouseout: 当鼠标移入 / 移出某个元素时触发。
  • mousemove: 当鼠标在某个元素上移动时触发。
  • mousedown / mouseup: 当鼠标按下 / 松开某个元素时触发。
  • dblclick: 当鼠标双击某个元素时触发。
JavaScript 键盘事件

用户在页面上输入文本时触发的事件。

  • keydown / keyup: 当按下 / 松开键盘上的键时触发。
  • keypress: 当键盘按键被按下并释放时触发。
JavaScript 表单事件

与表单元素相关的事件，如提交表单、输入框内容变化等。

  • submit: 当表单提交时触发。
  • change: 当表单元素的值发生改变时触发。
  • focus / blur: 当元素获得 / 失去焦点时触发。
JavaScript 窗口事件

与浏览器窗口相关的事件。

  • load: 当页面加载完成时触发。
  • resize: 当浏览器窗口大小发生改变时触发。
  • scroll: 当用户滚动页面时触发。
JavaScript 移动设备事件

针对移动设备上的触摸操作的事件。

  • touchstart / touchend / touchmove: 当用户触摸屏幕时触发的事件。
